argument which if free. When you finish filling click on “CREATE” button to save your task.

When you click on to the task you can see task in more details 

 

You can see tasks details by clicking on the task. At the top of the this dialog you can see 
tasks status: active, deleted, outdated, canceled or done. From this dialog you can make 
task done, cancel or delete. You can achieve this by clicking buttons from this dialog or 
clicking button from main screen. (see below image) 

 

Click on “EDIT” button if you want change something in task. If you save changes in task 
click on “UPDATE” button. 

 

When you click on “CANCEL” or “DONE” buttons the task disappear the main menu and 
you can find this task in “ALL TASKS” section. If you delete the task, it will goes to 
“RECYCLE BIN” section. 
 


If you want to know how much time you have to do the task click on time in task and it will 
swap with “time left” (see below) 

 

 

 

 

When you click on statistics section in main section you can get information about your tasks. 

 

 

Recycle Bin section 

When you delete task, it comes to recycle bin and here you can restore the task or delete the 
task deeply. You can that action by clicking on the task or clicking on the menu icon on the 
task (see below) 

 

All tasks section 

All tasks section has 4 pages: passed tasks, done tasks, canceled tasks, tasks have time. If the 
task is overdue, it will take place on passed tasks. If you mark the task done or canceled it 
take place on done tasks or canceled tasks. If the task have time to complete, you can find 
that task in tasks have time page. 

 


You can restore cancelled tasks here. If restored task has time to complete task is added to 
“tasks have time” page, otherwise you can find that in “passed tasks” page. 

 

In this section you can clone all task by clicking on “CLONE” button. Note!!! Only you can 
change task’s field, you can’t add new field when you clone the task.
